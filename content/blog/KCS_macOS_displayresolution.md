# Environment

macOS High Sierra+ / Mojave / BigSur

# Issue

You would like to easily set Mac Retina display(s) to higher unsupported resolutions.

# Resolution

- Free - [RDM](https://github.com/avibrazil/RDM)

- Shareware $14 - [SwitchResX](https://www.madrau.com/srx_download/download.html)

# Root Cause

macOS X only natively supports HiDPI modes on monitors with the most common Aspect Ratios.

More details - [Guide: Fixing External Monitor Scaling and 'Fuzziness' Issues with MBP and OSX](https://forums.macrumors.com/threads/guide-fixing-external-monitor-scaling-and-fuzziness-issues-with-mbp-and-osx.2179968/)
