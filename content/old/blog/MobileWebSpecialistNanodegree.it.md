Ho completato il percorso di studi **[Udacity Mobile Web Specialist Nanodegree](https://www.udacity.com/course/mobile-web-specialist-nanodegree--nd024)** (6-mesi - 10ore/settimana)!
Il corso e la certificazione è stata fornida dalla [borsa di studio Google Developer Scholarship EMEA](https://www.udacity.com/google-scholarships) (1.000 posti) dopo tre mesi di "Google Developer Challenge" (10.000 posti).

* [Udacity Mobile Web Specialist Nanodegree certificate.pdf](pdf/Udacity_Mobile_Web_Specialist_Nanodegree.pdf) (passed on 22<sup>th</sup> June 2018)

[<img src="img/Google-Dev-EMEA-Badge.png" width="25%" alt="Google Developer Scholarship">](https://www.udacity.com/google-scholarships)


Ho approfondito le mie conoscenze di sviluppo web front-end e back-end con le tecnologie HTML5, CSS3 (SASS), and JavaScript (ES2018) utilizzando lo strumento di automazione Gulp.
Sono inoltre in grado di eseguire Version Control con Git e GitHub.

Le principali abilità acquistie:

* Accessible and Responsive Web Apps

Ho lavorato passo passo alla realizzazione di una responsive web app che funzioni su qualsiasi dispositivo o dimensione dello schermo. Le mie App Web soddisfano i massimi livelli di sviluppo raggiungendo gli standard di accessibilità. Fondendo teoria e pratica ho imparato come creare fantastiche app mobile-first.

* Offline Capable Web Apps

Ho reso le mie app più resilienti alla connettività Internet o alla larghezza di banda limitata e crea un'esperienza utente totale sfruttando le funzionalità del browser asincrono, l'archiviazione locale ed altre funzionalità avanzate.

* Performant Web Apps

Ho esplorerato la pipeline di rendering del browser e scoperto i pattern che semplificano la creazione di app performanti. Ho ottimizzato le prestazioni della mia app Web ottimizzando il rendering, sfruttando l'archiviazione locale ed altro ancora.
