I completed the **[Udacity Mobile Web Specialist Nanodegree](https://www.udacity.com/course/mobile-web-specialist-nanodegree--nd024)** (6-months - 10hrs/week) nanodegree program!
The course and certification issued by [Google Developer Scholarship EMEA program](https://www.udacity.com/google-scholarships) (1.000 seats) after 3 months of Google Developer Challenge (10.000 seats).

* [Udacity Mobile Web Specialist Nanodegree certificate.pdf](pdf/Udacity_Mobile_Web_Specialist_Nanodegree.pdf) (passed on 22<sup>th</sup> June 2018)

[<img src="img/Google-Dev-EMEA-Badge.png" width="25%" alt="Google Developer Scholarship">](https://www.udacity.com/google-scholarships)


I learn a lot about front-end and back-end web development in HTML5, CSS3 (SASS), and JavaScript (ES2018) and the automator tool Gulp.
Also I'm proficient in Version Control with Git and GitHub.

Main skills acquired:

* Accessible and Responsive Web Apps

I worked step-by-step to build a responsive web app that functions on any device or screen size. Ensured my web apps met the highest development by achieving accessibility standards. Blending theory and practice, I learn how to build great apps to be mobile-first.

* Offline Capable Web Apps

I make my apps more resilient to limited internet connectivity or bandwidth, and create a total user experience leveraging asynchronous browser features, local storage, and more.

* Performant Web Apps

I explored the browser's rendering pipeline and uncover patterns that make it easy to build performant apps. I learn how to optimize my web app performance through rendering optimizations, leveraging local storage and more.
