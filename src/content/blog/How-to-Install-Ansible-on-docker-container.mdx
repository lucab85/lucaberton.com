Here is how to install Ansible inside a Docker container.

## Dockerfile

```dockerfile
FROM python:3.12-slim
RUN pip install ansible
RUN ansible-galaxy collection install community.general
WORKDIR /ansible
ENTRYPOINT ["ansible-playbook"]
```

## Build and Run

```bash
docker build -t ansible-runner .
docker run -v $(pwd):/ansible ansible-runner playbook.yml -i inventory
```

## Using Official Image

```bash
docker run --rm -v $(pwd):/ansible -w /ansible cytopia/ansible ansible-playbook playbook.yml
```
