Here is how to install Ansible on macOS.

## Install

```bash
brew install ansible
```

## Verify

```bash
ansible --version
```

## Alternative: pip

```bash
python3 -m venv ~/ansible-venv
source ~/ansible-venv/bin/activate
pip install ansible
```
