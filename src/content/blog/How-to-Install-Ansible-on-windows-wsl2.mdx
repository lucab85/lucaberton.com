Here is how to install Ansible on Windows using WSL2. Ansible does not run natively on Windows.

## Step 1: Enable WSL2

```powershell
wsl --install -d Ubuntu-24.04
```

## Step 2: Install Ansible

```bash
sudo apt update && sudo apt install -y ansible
```

## Verify

```bash
ansible --version
ansible localhost -m ping
```
