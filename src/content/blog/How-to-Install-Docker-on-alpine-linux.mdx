## Prerequisites

- Administrative/root access on Alpine Linux
- 64-bit system
- Internet connectivity

## Installation

### Step 1: Remove Old Versions

```bash
# Remove old Docker packages if present
sudo apt-get remove docker docker-engine docker.io containerd runc 2>/dev/null
```

### Step 2: Install Docker Engine

```bash
# Add Docker's official GPG key and repository
# Install Docker CE, CLI, containerd, and Compose plugin
```

### Step 3: Post-Installation

```bash
# Add your user to the docker group
sudo usermod -aG docker $USER
newgrp docker

# Enable Docker to start on boot
sudo systemctl enable docker
```

### Step 4: Verify

```bash
docker version
docker compose version
docker run hello-world
```

## Docker Compose

Docker Compose is included as a plugin:

```bash
# Verify Compose
docker compose version

# Example docker-compose.yml
cat > docker-compose.yml << 'EOF'
services:
  web:
    image: nginx:alpine
    ports:
      - "8080:80"
EOF

docker compose up -d
```

## Common Post-Install Issues

- **Permission denied**: Log out and back in after adding to docker group
- **DNS issues**: Configure `/etc/docker/daemon.json` with custom DNS
- **Storage**: Configure storage driver and log rotation
