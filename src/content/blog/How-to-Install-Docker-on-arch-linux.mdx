Here is how to install Docker on Arch Linux.

## Install

```bash
sudo pacman -Syu docker docker-compose
sudo systemctl start docker && sudo systemctl enable docker
sudo usermod -aG docker $USER
```

## Verify

```bash
docker run hello-world
```
