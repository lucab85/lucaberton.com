Here is how to install Docker on NixOS in 2026.

## Enable in configuration.nix

```nix
{
  virtualisation.docker.enable = true;
  users.users.youruser.extraGroups = [ "docker" ];
}
```

## Apply

```bash
sudo nixos-rebuild switch
```

## Verify

```bash
docker --version
docker run hello-world
docker compose version
```

## Temporary (nix-shell)

```bash
nix-shell -p docker
```
