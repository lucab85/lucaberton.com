Here is how to install Docker on Raspberry Pi.

## Install

```bash
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
newgrp docker
```

## Verify

```bash
docker run hello-world
```

**Note**: Use ARM64 images. Most official Docker images support `linux/arm64`.
