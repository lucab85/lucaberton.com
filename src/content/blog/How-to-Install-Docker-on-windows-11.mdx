Here is how to install Docker on Windows 11.

## Docker Desktop

Download from https://www.docker.com/products/docker-desktop/ — ensure WSL2 backend is selected.

## Without Docker Desktop (WSL2)

```bash
# Inside WSL2 Ubuntu
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
```

## Verify

```bash
docker --version
docker run hello-world
```
