## Prerequisites

Before installing Terraform on macOS Homebrew, ensure you have:

- Administrative/root access
- Internet connectivity
- A terminal or command prompt

## Installation Steps

### Step 1: Add the HashiCorp Repository

HashiCorp provides official packages for most platforms:

```bash
# Verify your system architecture
uname -m
```

### Step 2: Install Terraform

Follow the official installation method for macOS Homebrew:

```bash
# Download the latest Terraform binary
# Visit: https://developer.hashicorp.com/terraform/downloads
```

### Step 3: Verify Installation

```bash
# Check Terraform version
terraform version

# Expected output:
# Terraform v1.x.x
# on linux_amd64 (or your platform)
```

## Post-Installation Setup

### Configure Shell Autocompletion

```bash
# Install autocomplete
terraform -install-autocomplete

# Restart your shell
exec "$SHELL"
```

### Set Up Your First Project

```bash
# Create a project directory
mkdir terraform-demo && cd terraform-demo

# Create main.tf
cat > main.tf << 'EOF'
terraform {
  required_version = ">= 1.5.0"
}

output "hello" {
  value = "Terraform is working on macOS Homebrew!"
}
EOF

# Initialize and run
terraform init
terraform apply -auto-approve
```

## Managing Multiple Versions

For projects requiring different Terraform versions:

```bash
# Install tfenv (Terraform version manager)
git clone https://github.com/tfutils/tfenv.git ~/.tfenv
echo 'export PATH="$HOME/.tfenv/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

# Install specific version
tfenv install 1.7.5
tfenv use 1.7.5

# List installed versions
tfenv list
```

## Common Issues

### Permission Denied
```bash
# Fix binary permissions
chmod +x /usr/local/bin/terraform
```

### Command Not Found
```bash
# Add to PATH
echo 'export PATH="$PATH:/usr/local/bin"' >> ~/.bashrc
source ~/.bashrc
```

### Version Mismatch
```bash
# Check required version in your project
grep required_version *.tf

# Install matching version
tfenv install <version>
```
