## 🧱 What You’ll Learn Inside

This isn’t just documentation reworded. This is real-world know-how, packed into a step-by-step format.

---

### 🔍 Understanding the Architecture

- AAP 2.0’s container-based design  
- Automation Controller vs Automation Hub  
- Event-Driven Automation (EDA) and Automation Mesh  
- Execution Environments: what they are, why they matter  

---

### 🏗️ Installation & Setup – Made Simple

From basic installs to advanced configurations:

- Single-node setups with internal DBs  
- Multi-node with external databases  
- Load balancers, automation mesh, and redundancy strategies  
- Subscription activation, service management, backups, & logging  

> Yes, it’s complex — but Luca walks you through every option with diagrams, use cases, and tips.

---

### 🔐 Authentication, RBAC & Enterprise Integration

Configuring user roles shouldn’t feel like hacking a maze of LDAP or AD queries.

You'll learn how to:

- Set up secure **RBAC**  
- Integrate with **LDAP**, **Azure AD**, and more  
- Write clean queries for user groups  
- Automate team access without headaches  

---

### 🛠️ DevOps, GitOps, and Ansible AI

Modern automation isn't just about playbooks — it's about:

- Writing clean code using **Git workflows**  
- Leveraging **Ansible Lightspeed** (AI-assisted coding)  
- Speeding up development and reducing errors  
- Validating your Ansible content with quality dashboards  

> Luca even walks through integrating with upstream tools like **AWX**, explaining what’s compatible and what’s not.

---

### ☁️ Hybrid Cloud & Network Automation

If you're deploying to AWS, Azure, GCP, or using Kubernetes, this book has you covered.

You'll explore:

- Managing cloud workloads via AAP  
- Automating **Cisco** network backups  
- Interfacing with **Nutanix**, **Kubernetes**, and vendor-neutral infrastructures  
- Building reusable workflows that scale across environments  

---

### 🧩 Advanced Use Cases: Custom Modules, Plugins & Monitoring

Push your AAP skills to the next level:

- Create **custom plugins**, callback scripts, and lookup tools  
- Analyze automation quality using **Prometheus + Grafana**  
- Track execution metrics and system health  
- Build automation that runs itself — and tells you when it needs help
