After writing 8 books on Ansible, here is the distilled guide to doing it right.

## Prerequisites

- Ansible installed ([Install Ansible on Ubuntu](/blog/install-ansible-ubuntu-2404/) or [RHEL 9](/blog/install-ansible-rhel-9/))
- SSH access to target hosts
- Python 3 on managed nodes

## Step-by-Step Guide

### Step 1: Set Up Your Project

```bash
mkdir -p ansible-project/{inventory,roles,playbooks,group_vars}
cd ansible-project
```

### Step 2: Write Your First Playbook

```yaml
---
- name: Configure web servers
  hosts: webservers
  become: true

  tasks:
    - name: Install required packages
      ansible.builtin.dnf:
        name:
          - nginx
          - firewalld
        state: present

    - name: Start and enable services
      ansible.builtin.systemd:
        name: "{{ item }}"
        state: started
        enabled: true
      loop:
        - nginx
        - firewalld

    - name: Open HTTP port
      ansible.posix.firewalld:
        service: http
        permanent: true
        state: enabled
        immediate: true
```

### Step 3: Use Roles for Reusability

```bash
ansible-galaxy role init roles/webserver
```

## Best Practices

- Always use FQCN (fully qualified collection names)
- Use `ansible-lint` before every commit
- Test with Molecule
- Encrypt secrets with Ansible Vault
