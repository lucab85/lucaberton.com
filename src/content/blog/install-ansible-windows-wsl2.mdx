Here is how to install Ansible on Windows using WSL2 in 2026. Ansible does not run natively on Windows — WSL2 is the recommended approach.

## Step 1: Enable WSL2

```powershell
# Run in PowerShell as Administrator
wsl --install
# Restart your computer
```

## Step 2: Install Ubuntu in WSL2

```powershell
wsl --install -d Ubuntu-24.04
```

## Step 3: Install Ansible Inside WSL2

```bash
# Update packages
sudo apt update && sudo apt upgrade -y

# Install Ansible
sudo apt install -y ansible

# Or via pip for latest version
sudo apt install -y python3-pip python3-venv
python3 -m venv ~/ansible-venv
source ~/ansible-venv/bin/activate
pip install ansible
```

## Verify

```bash
ansible --version
ansible localhost -m ping
```

## Access Windows Files from WSL2

```bash
# Windows C: drive is mounted at /mnt/c
ls /mnt/c/Users/YourUsername/
```
