Here is how to install Docker on macOS in 2026. Tested and verified.

## Option 1: Docker Desktop

Download from https://www.docker.com/products/docker-desktop/ and follow the installer.

```bash
# Verify
docker --version
docker compose version
docker run hello-world
```

**Note**: Docker Desktop requires a paid subscription for companies with 250+ employees or over $10M revenue.

## Option 2: Podman Desktop (Free Alternative)

```bash
brew install podman
podman machine init
podman machine start

# Docker-compatible alias
alias docker=podman

# Verify
podman run hello-world
```

## Option 3: Colima (Lightweight)

```bash
brew install colima docker
colima start

# Uses Docker CLI with Colima backend
docker run hello-world
```

## Post-Installation

```bash
# Resource configuration (Docker Desktop)
# Settings > Resources > CPUs: 4, Memory: 8 GB

# Or for Colima
colima start --cpu 4 --memory 8
```
