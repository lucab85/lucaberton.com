Here is how to install kubectl on RHEL 9. Tested and verified.

## Prerequisites

- RHEL 9 installed and updated
- Root or sudo access
- Internet connection

## Step 1: Update System

```bash
sudo dnf update
```

## Step 2: Add Official Repository

kubectl should be installed from the official repository to get the latest stable version and security updates.

```bash
# Add the official kubectl repository
# Check kubectl documentation for the latest repo URL
sudo dnf install -y curl gnupg2
```

## Step 3: Install kubectl

```bash
sudo dnf install -y kubectl
```

## Step 4: Verify Installation

```bash
kubectl version
```

Expected output should show the installed version.

## Step 5: Post-Install Configuration

```bash
# Enable and start the service (if applicable)
sudo systemctl enable --now kubectl

# Verify status
sudo systemctl status kubectl
```

## Troubleshooting

### Package Not Found

Make sure the repository was added correctly:

```bash
# List configured repositories
dnf repolist
```

### Permission Denied

Ensure your user has the correct group membership:

```bash
sudo usermod -aG kubectl $USER
newgrp kubectl
```

## Uninstall

```bash
sudo dnf remove -y kubectl
```
