Here is how to install Kubernetes on Fedora 41. Tested and verified.

## Prerequisites

- Fedora 41 installed and updated
- Root or sudo access
- Internet connection

## Step 1: Update System

```bash
sudo dnf update
```

## Step 2: Add Official Repository

Kubernetes should be installed from the official repository to get the latest stable version and security updates.

```bash
# Add the official Kubernetes repository
# Check Kubernetes documentation for the latest repo URL
sudo dnf install -y curl gnupg2
```

## Step 3: Install Kubernetes

```bash
sudo dnf install -y kubernetes
```

## Step 4: Verify Installation

```bash
kubernetes version
```

Expected output should show the installed version.

## Step 5: Post-Install Configuration

```bash
# Enable and start the service (if applicable)
sudo systemctl enable --now kubernetes

# Verify status
sudo systemctl status kubernetes
```

## Troubleshooting

### Package Not Found

Make sure the repository was added correctly:

```bash
# List configured repositories
dnf repolist
```

### Permission Denied

Ensure your user has the correct group membership:

```bash
sudo usermod -aG kubernetes $USER
newgrp kubernetes
```

## Uninstall

```bash
sudo dnf remove -y kubernetes
```
