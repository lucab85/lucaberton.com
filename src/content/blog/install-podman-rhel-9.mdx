Here is how to install Podman on RHEL 9. Tested and verified.

## Prerequisites

- RHEL 9 installed and updated
- Root or sudo access
- Internet connection

## Step 1: Update System

```bash
sudo dnf update
```

## Step 2: Add Official Repository

Podman should be installed from the official repository to get the latest stable version and security updates.

```bash
# Add the official Podman repository
# Check Podman documentation for the latest repo URL
sudo dnf install -y curl gnupg2
```

## Step 3: Install Podman

```bash
sudo dnf install -y podman
```

## Step 4: Verify Installation

```bash
podman version
```

Expected output should show the installed version.

## Step 5: Post-Install Configuration

```bash
# Enable and start the service (if applicable)
sudo systemctl enable --now podman

# Verify status
sudo systemctl status podman
```

## Troubleshooting

### Package Not Found

Make sure the repository was added correctly:

```bash
# List configured repositories
dnf repolist
```

### Permission Denied

Ensure your user has the correct group membership:

```bash
sudo usermod -aG podman $USER
newgrp podman
```

## Uninstall

```bash
sudo dnf remove -y podman
```
