Here is how to install Prometheus on Ubuntu in 2026. Tested and verified.

## Prerequisites

- A Ubuntu system with root or sudo access
- Internet connectivity
- At least 2 GB RAM (4 GB recommended)

## Installation Steps

### Step 1: Update System Packages

```bash
# Update package index
sudo apt update && sudo apt upgrade -y  # Debian/Ubuntu
# or
sudo dnf update -y  # RHEL/Fedora
```

### Step 2: Install Prometheus

Follow the official installation method for Ubuntu. Check the [Prometheus documentation](https://kubernetes.io/docs/) for the latest instructions.

### Step 3: Verify Installation

```bash
# Check version
prometheus version

# Verify the installation works
prometheus --help
```

## Post-Installation Configuration

- Configure firewall rules if needed
- Set up auto-completion for your shell
- Create a non-root user for running Prometheus
- Configure logging and monitoring

## Troubleshooting

If you encounter issues:

1. Check system logs: `journalctl -xeu prometheus`
2. Verify system requirements are met
3. Check firewall rules
4. Review the [troubleshooting section](/blog/) for common errors
