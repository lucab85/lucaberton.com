## What You'll Learn

- ✅ Create DAGs with default_args, schedule, and tags
- ✅ Define tasks using TaskFlow API and classic operators
- ✅ Wire dependencies that Airflow can use to form graphs
- ✅ Configure retries and alert plugins
- ✅ Enable DAGs, trigger runs, and execute backfills
